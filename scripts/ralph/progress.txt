## Codebase Patterns
- iOS Native 是主线（DayPage/），Web 是跟进（web/）；本 PRD 不动数据层/auth/proxy，只换 UI 与 +7 endpoints。
- 本地 Web dev 必须同时跑 `npm run dev`（3000）与 `npm run dev:inngest`（8288），否则 memo 入库但不编译。
- 设计 token 是单一来源：先做 US-001（design-tokens/tokens.json + 生成器）再做任何 UI story，避免颜色 hardcode。
- 所有交互组件需引用 `var(--*)` token，不要写死颜色 / 字号 / 圆角。
- 手势 0 依赖原则：不引入 framer-motion / use-gesture / react-spring；用 pointer events + CSS transition + cubic-bezier(.2,.8,.2,1) spring。
- Memo Card 左滑公式（rubber-band 阻尼 0.18 / 阈值 -66 / 过界 -164）见 PRD §22 Appendix A.1，多个 sheet 复用。
- 所有动画必须 wrap @media (prefers-reduced-motion: reduce) 退化为 0.01ms，否则 a11y 不过。
- iOS Live Activity（US-030）需要 deployment target bump 16.0 → 16.1，需 Eric 确认（PRD Open Q1）。
- Share Card 导出（US-033）跨域图片必须走 /api/img/{memo_id} same-origin proxy，html-to-image 才能成功。
- 新 endpoints 全部在 web/src/app/api/today/ 与 web/src/app/api/stats/ 下，schema 见 PRD §8 + fixtures §24。
- iOS 端构建命令: `xcodebuild -scheme DayPage -destination 'platform=iOS Simulator,name=iPhone 17'`；Simulator 用 `open -a Simulator &` 起后台不阻塞。
- 字体新增：Fraunces（serif）与 Caveat（手写）— iOS 进 bundle + Info.plist 注册；Web 用 next/font 配置。
- Drawer 热力图（US-021）若 user memo 总数 > 5000 回退读 users.heatmap_cache_json，由 cron 6h 预计算（避免每次 group by）。
